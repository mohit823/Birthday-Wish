# Birthday Wish for Mohit

A delightful and interactive birthday wish webpage created for Mohit, featuring beautiful animations, balloons, confetti, fireworks, and more!

## Features

- **Animated Background**: Shifting gradient colors for a dynamic feel
- **Floating Balloons**: Colorful balloons floating up continuously
- **Falling Confetti**: Random confetti pieces falling from the top
- **Interactive Cake**: Bouncing cake emoji with click-to-cut feature
- **Personalized Message**: Heartfelt birthday message in a styled card
- **Special Traits Section**: Highlights what makes Mohit special
- **Photo Gallery**: Placeholder images for birthday memories
- **Fun Quiz**: Interactive quiz about Mohit with multiple questions
- **Wish Button**: Triggers fireworks and a final popup message
- **Sound Toggle**: Enable/disable sound effects for fireworks
- **Music Toggle**: Play/pause background birthday music
- **Dark Mode Toggle**: Switch between light and dark themes
- **Share Buttons**: Share the page on Facebook, Twitter, and WhatsApp
- **Birthday Facts**: Rotating fun facts about birthdays
- **Message Form**: Leave personalized birthday messages (emailed to creator + saved locally)
- **Countdown Timer**: Countdown to next birthday
- **Floating Hat**: Animated birthday hat decoration
- **Gift Box**: Clickable gift that reveals a surprise message
- **Photo Slideshow**: Auto-rotating memory slideshow
- **Snowfall Effect**: Continuous snow animation
- **Color Customizer**: Change balloon and confetti colors
- **Memory Game**: Match pairs of birthday-themed emojis
- **Typing Effect**: Animated typing of the birthday name
- **Click Particles**: Click anywhere to create sparkles and hearts
- **Fireworks Display**: Canvas-based fireworks with particles
- **Mouse Particles**: Hearts and sparkles follow mouse movement
- **Responsive Design**: Works beautifully on mobile, tablet, and desktop
- **Smooth Animations**: CSS animations for a polished experience

## Email Setup

Messages submitted through the form are automatically emailed to the creator. To enable this feature:

1. Create a free account at [EmailJS](https://www.emailjs.com/)
2. Set up an email service and template
3. Update the API keys in `index.html`
4. See `EMAIL_SETUP.md` for detailed instructions

## How to View

Simply open `index.html` in any modern web browser. No server required!

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- JavaScript (ES6+)
- Canvas API for fireworks
- EmailJS for message delivery
- Local Storage API for message backup

## Browser Support

Works best in modern browsers that support CSS animations and Canvas API (Chrome, Firefox, Safari, Edge).
